# Code Book
***
This book explains the output from tidydata.txt

Ouptput contains the average of the measurment variables organized by activities and subjects. 

> Identifiers:

`subject` from 1 to 30. Individuals who participate the data collections. 

`acti_d` activities taken for the data collections. This is in desrcriptive forms, rather than numerical form. 

There're six activities in total:

1. WALKING

2. WALKING_UPSTAIRS

3. WALKING_DOWNSTAIRS

4. SITTING

5. STANDING

6. LAYING
